# ewed - AI Celebrity Marriage Certificate Generator

This is a fun web application that allows you to create a personalized, placeholder marriage certificate with yourself and your favorite celebrity.

## Features

- **Live Preview:** See your certificate update in real-time as you customize it.
- **AI-Powered Content:** Generates a unique celebrity portrait and a witty witness statement from Batman using the Gemini API.
- **Customization:** Upload your own photo, write custom vows, and choose from different font styles.
- **Download & Print:** Save your final certificate as a high-quality PNG image.
